Coloring Blobs¶

Since update 1.6.3, all blobs can now be colorized. The capsule blob is unique because it has separate colors for the root and tip.
Enabling the Colorize Blobs button will generate the BlobColor attribute. This feature has performance cost of approximately 10-20% slower SDF calculations. You can disable it when not needed.

Demo of colorizing Blobs, and gradient fill tool

Enable Colorize Blobs button, to see the color properties of selected blobs. If you can't see it try updating your blend file nodes using:

  • Blob Panel > Gear Icon > Update Blobs

Background Color¶

When two blobs are mixed together, with high blending factor, the background color will be visible. This can be used to create interesting effects.

You can see the white color in places where blobs meet together. This is due to 'Backgrond Color' being white by default
